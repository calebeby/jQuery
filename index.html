What is jQuery?

jQuery is a JavaScript library that makes selecting elements easier.

jQuery:

$(".element").addClass("test");

JavaScript:

document.getElementByClass("element").classList.add("test");

Implementation:

You can use it by including the jQuery files, either by CDN or hosting it yourself:

Google CDN

<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.0/jquery.min.js"></script>

Things you can do with jQuery:

AJAX Calls:

$.ajax({ url: "demo_test.txt", success: function(result){ $("#div1").html(result); } });
